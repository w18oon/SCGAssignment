# SCG Assignment
## How to run
1. Git Clone
``` bash 
git clone https://github.com/w18oon/SCGAssignment.git
```
2. Updating dependencies
``` bash
composer update
```
3. Install node package
``` bash
npm install && npm run dev
```
4. Serving Laravel
``` bash
php artisan serve
```
5. Testing
> Open browser and go to http://127.0.0.1:8000
